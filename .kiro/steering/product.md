# Product Overview

## AIGC 沉浸式个人履历门户

A high-performance, visually stunning portfolio and dynamic resume portal designed specifically for AIGC creators and developers, featuring 3D parallax effects and deep tech aesthetics.

### Core Purpose
- **Personal Portfolio**: Showcase AIGC-generated content (images, videos, 3D renders)
- **Professional Resume**: Display work experience, skills, and project achievements
- **Interactive Gallery**: Immersive media viewing with advanced lazy loading and modal systems
- **Responsive Design**: Optimized for all devices with print-friendly layouts

### Target User
任泓雨 (Ren Hongyu) - Information technology project delivery specialist with 3-4 years experience in large-scale government projects, passionate about AI/AIGC tools and applications.

### Key Features
- **Dual Theme System**: Light/dark mode with smooth transitions
- **Media Gallery**: Advanced image/video gallery with lazy loading, thumbnails, and modal viewing
- **3D Parallax Effects**: Immersive visual experience using Framer Motion
- **Performance Optimized**: Lazy loading, compressed media, and efficient rendering
- **Print Support**: Professional resume export functionality
