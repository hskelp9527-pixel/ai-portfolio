# 图片资源目录说明

## 目录结构

- **original/**: 存放原始高清图片文件
  - 建议格式：JPG, PNG, WebP
  - 建议尺寸：1920x1080 或更高
  - 文件大小：建议不超过 5MB

- **compressed/**: 存放压缩版本图片（用于网页显示）
  - 建议尺寸：1200x900 或适当比例
  - 文件大小：建议 500KB - 2MB
  - 格式：WebP 优先，JPEG 备选

- **thumbnails/**: 存放缩略图（用于预览和懒加载）
  - 建议尺寸：400x300
  - 文件大小：建议 50KB - 200KB
  - 格式：WebP 或 JPEG

## 文件命名规范

- 使用英文和连字符，避免空格和特殊字符
- 例如：`future-city-aigc.jpg`
- 保持三个目录中的文件名一致（仅扩展名可能不同）

## 使用说明

1. 将你的原始图片放入 `original/` 目录
2. 创建压缩版本放入 `compressed/` 目录
3. 创建缩略图放入 `thumbnails/` 目录
4. 在 `data.ts` 中更新 `IMAGES` 数组，添加对应的文件信息
